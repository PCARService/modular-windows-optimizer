Traducciones: kury_vandham (pt)

ESTE CONJUNTO DE SCRIPTS REALIZA OPTIMIZACIONES A NIVEL DEL SISTEMA OPERATIVO.
(Windows 10 / 11). NO REALIZA MODIFICACIONES SOBRE BIOS/UEFI NI FIRMWARE.

¿Cómo se utiliza?
- Abrir el archivo "Run.cmd" con derechos de Administrador.
- Esperar a que finalice la optimización.
- Reiniciar PC.
- Eso es todo. :b

Opcional:
El archivo 'try this after.reg' puede utilizarse como refuerzo manual
para desactivar ciertos servicios que Windows puede reactivar automáticamente.
No es obligatorio para el funcionamiento del script principal.

Modular Set for System Optimization (MSSO)
Designed and tested by: Mathero
Translations: kury_vandham (pt)

THIS TOOL APPLIES OPERATING SYSTEM-LEVEL OPTIMIZATIONS
(Windows 10 / 11 only). IT DOES NOT MODIFY BIOS/UEFI OR FIRMWARE SETTINGS.

How to use?
- Open "Run.cmd" with Administrator rights.
- Wait until optimization ends.
- Restart PC.
- That's it. :b

Optional:
The 'try this after.reg' file can be used as a manual reinforcement
to disable certain services that Windows may re-enable automatically.
It is not required for normal operation.
